RecorderX is a high-performance, privacy-focused screen recorder for Android.

Key Features:
- Record up to 4K (UHD) and 2K (QHD) for crystal clear video.
- High-speed recording at 60fps, 90fps, and 120fps.
- Custom bitrates up to 40 Mbps for professional quality.
- Modern AMOLED Black and Lavender Purple theme with precision slider controls.
- Automatic file naming with custom templates.
- Self-healing audio system that recovers if the system blocks capture.
- Post-recording notifications with live video thumbnails.
- Works entirely offline with no internet access or ads.
- Open Source and built for performance.
